Paginate
filter in module rxDataTable

Description

This is the pagination filter that is used to calculate the division in the items list for the paging.

Usage

In HTML Template Binding

{{ Paginate_expression | Paginate:pager-object }}

In JavaScript

$filter('Paginate')(items, pager-object)

Parameters

ParamTypeDetails
itemsObject

The list of items that are to be sliced into pages

pager-objectObject

The instance of the PageTracking service. If not specified, a new one will be created.

Returns

Object

The list of items for the current page in the PageTracking object
