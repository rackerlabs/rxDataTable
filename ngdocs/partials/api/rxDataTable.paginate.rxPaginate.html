rxPaginate
directive in module rxDataTable

Description

Directive that takes in the page tracking object and outputs a page switching controller

Usage

as element:
<rx-paginate
       page-tracking="{Object}"
       number-of-pages="{number}">
</rx-paginate>

Parameters

ParamTypeDetails
page-trackingObject

This is the page tracking service instance to be used for this directive

number-of-pagesnumber

This is the maximum number of pages that the page object will display at a time.
